PDF Footnotes Processor

Extract and process footnotes from PDF documents

Features

  • Automatic footnote detection and extraction
  • Maintains text formatting and structure
  • Generates formatted Excel output
  • Real-time processing status updates

Documentation

Installation

git clone https://github.com/laxman824/PDF-Footnotes-processed.git
cd PDF-Footnotes-processed
pip install -r requirements.txt

Usage

  1. Upload your PDF file
  2. Click "Process PDF"
  3. Download the generated Excel file
